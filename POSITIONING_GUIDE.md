# Руководство по изменению позиционирования контейнеров

## Как изменить позиционирование и отступы

### 1. Основные CSS переменные (файл: src/index.css)

Все настройки позиционирования находятся в CSS переменных в начале файла `src/index.css`:

```css
:root {
  /* Вертикальные отступы между контейнерами */
  --container-spacing: 1rem; /* 16px - основной отступ между всеми контейнерами */
  
  /* Горизонтальные отступы для кнопок дублирования */
  --duplicate-button-spacing: 0.75rem; /* 12px - отступ кнопок дублирования */
  
  /* Ширина всех контейнеров */
  --container-width: 400px; /* ширина всех контейнеров */
  
  /* Дополнительные отступы для отдельных контейнеров */
  --session-tracker-top-margin: 0;      /* отступ сверху для трекера сессий */
  --session-manager-top-margin: 0;      /* отступ сверху для менеджера сессий */
  --theory-tracker-top-margin: 0;       /* отступ сверху для трекера теории */
  --theory-session-manager-top-margin: 0; /* отступ сверху для менеджера теории */
}
```

### 2. Как изменить отступы

#### Изменить основной отступ между всеми контейнерами:
```css
--container-spacing: 2rem; /* увеличить до 32px */
--container-spacing: 0.5rem; /* уменьшить до 8px */
```

#### Изменить отступ для конкретного контейнера:
```css
--session-tracker-top-margin: 1rem; /* добавить 16px сверху для трекера сессий */
--theory-tracker-top-margin: 2rem;  /* добавить 32px сверху для трекера теории */
```

#### Изменить отступ кнопок дублирования:
```css
--duplicate-button-spacing: 1rem; /* увеличить отступ кнопок до 16px */
```

#### Изменить ширину всех контейнеров:
```css
--container-width: 500px; /* увеличить ширину до 500px */
--container-width: 350px; /* уменьшить ширину до 350px */
```

### 3. Примеры изменений

#### Увеличить отступы между контейнерами:
```css
:root {
  --container-spacing: 2rem; /* 32px вместо 16px */
}
```

#### Добавить дополнительный отступ сверху для трекера теории:
```css
:root {
  --theory-tracker-top-margin: 1rem; /* добавить 16px сверху */
}
```

#### Увеличить ширину всех контейнеров:
```css
:root {
  --container-width: 450px; /* увеличить с 400px до 450px */
}
```

#### Увеличить отступ кнопок дублирования:
```css
:root {
  --duplicate-button-spacing: 1rem; /* увеличить с 12px до 16px */
}
```

### 4. Единицы измерения

Вы можете использовать любые CSS единицы:
- `rem` - относительно размера шрифта (1rem = 16px по умолчанию)
- `px` - пиксели
- `em` - относительно размера шрифта родителя
- `%` - проценты
- `vh/vw` - проценты от высоты/ширины экрана

### 5. Применение изменений

После изменения CSS переменных:
1. Сохраните файл `src/index.css`
2. Изменения применятся автоматически благодаря Hot Module Replacement
3. Если изменения не применились, обновите страницу в браузере

### 6. Сброс к исходным значениям

Чтобы вернуть исходные настройки:
```css
:root {
  --container-spacing: 1rem;
  --duplicate-button-spacing: 0.75rem;
  --container-width: 400px;
  --session-tracker-top-margin: 0;
  --session-manager-top-margin: 0;
  --theory-tracker-top-margin: 0;
  --theory-session-manager-top-margin: 0;
}
```
